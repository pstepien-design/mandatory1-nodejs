GETTING STARTED

NodeJS is a run time environment that allows to run javascript outside of the browser. It utilises V8 engine created by Google. NodeJS allows for server-side scripting, which is running the script before the website is send to the web browser.

HOW TO START

To initialize NodeJS project we need to install NodeJS on our computer, along with the NPM. NPM stands for Node Package Manager and allows us to install and modify packages inside of our project.
After installing both things, we can create a folder in which we will place our project. Then we just need to type npm init to initialize the project. Then we will be asked to fill out the information about the package name, version, description, entry point(which is our main java script file like app.js), test, git repository, keywords, author and license we want to utilise. Then we need to confirm the details we provided and we are done with project initialization!
To work easily with the code, we can go to the terminal and move to our project's directory. Then by typing code . we can open our file in VS Code compilator (this feature needs to be installed first in out vs code).

GETTING STARTED WITH JAVASCRIPT

FUNCTIONS

In Java Script we have two different types of fucntions: normal function and arrow function.
Normal function is declared by the function keyword, follow by the name parantheses and curly brackets
function myFunction() { ... }
Arrow function is a bit different. It is assigned to a variable.
const myfunction = () => { .. }

VARIABLES

In JavaScript objects look similar to json objects
const meme = { points: 54234, };
We can access object's variable or add a new one by just refering to the object's name, dot and variable name
meme.points += 1; meme.age = 'old';

REST API

While creating APIs, which are protocols for building and integrating software, we can utilise the rules of REST. Request are managed through HTTP requests such as:

POST

To add a new item

DELETE

To delete an item

GET

To get an item/ a list of items

PATCH

Updating only some details about the item

PUT

Updating all the details about the item
