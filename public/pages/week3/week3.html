LOOPS

We got used to working with loops, when we want to for example get the value of every object in the array. However, the typical, "old-school" loop takes lots of lines of code:
Let's say we have an array of numbers:
const numbers = [1, 2, 3, 4];
If we want to increment each value of the array, the loop would look like this.
for(let number of numbers){ number += 1; }
There is a faster solution. We can use forEach method.
numbers.forEach(number => number++);
However, forEach has one limitation - it does not return anything, it can only iterate. If we want to return a value, the better solution is to use .map function.
numbers.map(number => { number++ return number; });

EXPRESS -> BACKEND SERVER

Node Package Manager - mostly known as NPM. It's purpose is to put packages in the place, so node can find them and manages dependencies. To install node modules we just need one command in the terminal
npm install
This way we have installed node modules into our project. What we can also do it install dependencies such as express, nodemon or bootstrap. We just simply need to type:
npm install express
Another way to install dependencies is by specifying them in the package.json file.
"dependencies": { "express": "^4.17.3" }
To implement imports such as express in our app.js file we have two options:
1. The newer one is using import:
import express from 'express';
2. The other one is using require:
const express = require('express');
