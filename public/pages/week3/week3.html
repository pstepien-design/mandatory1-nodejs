LOOPS

We got used to working with loops, when we want to for example get the value of every object in the array. However, the typical, "old-school" loop takes lots of lines of code:
Let's say we have an array of numbers:
const numbers = [1, 2, 3, 4];
If we want to increment each value of the array, the loop would look like this.
for(let number of numbers){ number += 1; }
There is a faster solution. We can use forEach method.
numbers.forEach(number => number++);
However, forEach has one limitation - it does not return anything, it can only iterate. If we want to return a value, the better solution is to use .map function.
numbers.map(number => { number++ return number; });

EXPRESS -> BACKEND SERVER

As it was mentioned before we can utilise express in node. Express allows us to process different http requests on our server.
First we need to import express, after that we need to specify our application will use it. Then we just need to write app. followed by the desired request.
const express = require('express');
const app = express();
app.get('/', (req, res) => { res.send('getting') });
app.post('/', (req, res) => { res.send('posting') });
To be able to access the server we need to specify the port it will work on. We utilise app.listen for this.
app.listen(8080, () => { console.log('The server is running', 8080); });
If we want to work with different files in our project structure, we should utilise express.static By doing this, we specify the directory of the files to serve.
app.use(express.static(__dirname + '/public'));
__dirname is a variable that is used to determine the absolute path of the file.
